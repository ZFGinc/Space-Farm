# Описание
На земле появился инопланетный монстр, замаскированный под простой магазин овощей и фруктов. Каждые N дней он становится всё голоднее и требует всё новую еду для уталения голода. Если его не накормить определенным количеством еды, то он начнет поглащять землю, что приведет к проигрышу. Так же он может выдавать задания на посещение других планет и запросы новых инопланетных культур с этих планет.
Монстр дает специальный механизм для путешествия между планетами для выполнения его заданий.

Игроку придется вырыщивать различные культуры как земного, так и внеземного происхождения для кормления монстра. Путешествовать по другим планетам и обустраивать там фермы для выращивания урожая. Все планеты пригодны для выращивание только определенных культур. 

На каждой планете нужно будет первым делом установить кислородный купол (он всегда 1 уровня). По мере улучшения кислорожного купола на этой планете увеличивается область на которой можно работать.
Если на планете присутствуют хищные растения, то для начала нужно будет от них избавиться. После убийств этих растений, из них можно получить семена, однако перед началом засеивания новооткрытой культуры, ее нужно исследовать на специальном аппарате, чтобы сделать ее безопастной. 
Так же можно эту культуру улучшить с помощью ГМО для увеличения количества урожая, уменьшения времени роста и др.

Для автоматизации будут доступны автополивалки, автосборщики урожая, автозасеиватели и др. Всё это можно будет построить и улучшить.

Есть возможность играть вместе с друзьями по Steam - максимум 4 игрока.

# Системы и механики
## Перемещение и поворот игрока
Спроектировать архитектуру для удобного переключения устройства ввода.

Перемещение игрока будет осуществляться с помощью CharacterController.

*Псевдо-курсор (далее просто ПсК)* - это точка/область направленная на карту. Он перемещается с помощью *ПовИ* по карте.
Поворот персонажа будет осуществляться в сторону *ПсК* для всех устройств.

## Взаимодействие игрока с объектами
Игрок может взаимодействовать с другими игровыми объектами в радиусе [Пар.Игр.4](#управление).
Если *ПсК* выходит за этот радиус, то он меняет свой цвет, показывая невозможность взаимодействия игрока с объектом.

Если игрок нажмет кнопку [Исп](#управление), то будет произведено действие с объектом, на который наведен *ПсК*
Если игрок нажмет кнопку [СОП](#управление), то будет произведено действие СХВАТИТЬ отбъект на который наведен *ПсК*, либо ОТПУСТИТЬ объект, который держал персонаж.

## Инвентарь и хотбар
С помощью [ВСХ](#управление) можно выбрать активный слот хотбара.
В [параметрах](#параметры), пункт 5 указан размер хотбара.

На кнопку [Инв](#управление) будет открываться/закрываться инвентарь игрока. В [параметрах](#параметры), пункт 6 указан размер инвентаря. С помощью [ПовИ](#управление) будет выбор предмета. Предмет на который наведен [ПовИ](#управление) будет подсвечен.

Если игрок нажмет [ВПИИ](#управление) предмет будет выброшен из инвентаря.

Если игрок нажмет [Исп](#управление) предмет будет выбран в активный слот хотбара. Если пустых слотов хотбара нет, то будет выдано предупреждение, что у персонажа заняты руки.

Если игрок нажмет кнопку [СОП](#управление), то будет произведено действие СХВАТИТЬ отбъект на который наведен *ПсК* - он будет положен в активный или ближайший свободжны слот хотбара. 
Если пустых слотов хотбара нет, то он сразу положит его в инвентарь, но если инвентарь игрока так же полон, то будет выдано предупреждение, что у персонажа закончилось свободное место.
Если игрок нажмет кнопку [СОП](#управление) и при этом он будет держать предмет в активноп слоте хотбара, то будет произведено действие ОТПУСТИТЬ предмет, который держал персонаж в руках, а предмет удалится из хотбара игрока и выпадет на землю.

Исключения:
  * У предметов будет настраиваться правила для возможностей выкидывания (например базовые предметы/оружия нельзя будет выкинуть из инвентаря).

## Строительство
Будет осуществляться с помощью *ПсК* для установки на нужное место и кнопки [ПодтСтр](#управление) для установки объекта.
Что бы повернуть объект перед установкой, нужно нажать клавишу [ПовОПС](#управление). Данная клавиша поворачивает объект на [Пар.Стр.2](#параметры) градусов

## Сбор квоты
На стартовой планете будет находиться магазин, куда можно продавать выращенную культуру. Каждый игровой день, цены будут меняться в зависимости от того, сколько вы продали того или иного товара в прошлые дни.
Магазин жаден, поэтому ему нужно отдавать всё больше и больше разных культур разного количества.

## Фермерство
С помощью мультитула можно вспахивать землю и формирования грядок для дальнейшего засеивания. Вырубать ненужные деревья.

Для засеивания новых культур нужно будет иметь в активном слоте хотбара семена нужной культуры - наведя *ПсК* и нажав [Исп](#управление) будет произведен процесс засеивания культуры.

Для поливки грядок будет использоваться Распылитель. В зависимости от уровня будут дополнительные возможности.

## Завоевываение новых территорий
Монстр дает информацию о новой планете и требует определенный ресурс от туда. В списке планет механизма для путешествий появится новая планета, на которую можно настроить механизм.
На новой планете могут обитать как мирные растения, так и опасные.
